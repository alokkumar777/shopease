# ShopEase - Modern E-commerce Prototype

A full-stack e-commerce application built with React (Frontend) and Django (Backend).

## Features
- **Product Browsing:** Explore featured products with a polished UI.
- **Cart Management:** Add products to cart and manage quantities.
- **Order Lifecycle:** Complete checkout and track order history.
- **Dynamic Order Status:** Simulated real-time delivery status updates.
- **Modern UI:** Styled with Bootstrap 5 and custom polished themes.

## Tech Stack
- **Frontend:** React, React Router, Axios, Bootstrap 5.
- **Backend:** Django, Django Rest Framework, SimpleJWT.
